## 🐛 НОВЫЕ БАГИ (обнаружены при тестировании 2026-04-14)

### Bug #1: Warning о сне < 7 часов не виден
**Симптом:** При установке интервала сна < 7 часов предупреждение появляется внизу модального окна и не видно
**Ожидание:** Предупреждение должно быть поверх окна настроек (overlay/modal поверх modal)
**Приоритет:** 🔴 Высокий — пользователь не видит ошибку

### Bug #2: Язык интерфейса меняется только после перезагрузки
**Симптом:** При смене языка в Settings → Save Changes язык меняется только после F5
**Ожидание:** Язык должен переключиться сразу после нажатия "Save Changes"
**Приоритет:** 🔴 Высокий — ломает UX

### Bug #3: Двойная кавычка в уведомлении о переносе
**Симптом:** `"Зазада"тестовая задача" перенесена на Утро завтра 15 апреля"`
**Причина:** Возможно баг в `.replace('{title}', newTaskTitle)` — лишняя буква "За" + кавычка
**Приоритет:** 🟡 Средний — косметический баг

### Bug #4: Update команды не работают
**Симптом:** "Добавь задачу один" → `Command confidence: 0.64 Type: add_task` — вместо add_task должен быть правильный тип
**Приоритет:** 🔴 Высокий

**Приоритет:** 🔴 Высокий (Accessibility)

#### Подготовка
- [ ] Открыть Settings → найти секцию Accessibility / Notifications
- [ ] Переключатель "Visual notifications" должен быть

#### Тест А: Визуальное подтверждение голосовой команды
- [ ] Включить "Visual notifications"
- [ ] Нажать 🎤 микрофон
- [ ] Сказать: "Добавь задачу тест"
- [ ] **Проверить:**
  - [ ] Вместо TTS "Задача добавлена" — визуальный toast/badge
  - [ ] Toast виден хорошо (контрастный, не прозрачный)
  - [ ] Toast исчезает через 2-3 секунды

#### Тест Б: Визуальный будильник
- [ ] Включить "Visual notifications"
- [ ] Установить будильник на 1 минуту вперёд
- [ ] Подождать срабатывания
- [ ] **Проверить:**
  - [ ] Вместо звука — пульсирующий экран (тёплый цвет)
  - [ ] Full-screen визуальный сигнал
  - [ ] Кнопка "Snooze" и "Stop" видны

#### Тест В: prefers-reduced-motion
- [ ] В DevTools → Rendering → Emulate CSS media feature → `prefers-reduced-motion: reduce`
- [ ] **Проверить:**
  - [ ] Пузыри не "дышат" — статичные круги
  - [ ] Физика пузырей отключена
  - [ ] Плавное fade-in вместо анимации

#### Тест Г: prefers-contrast: more
- [ ] В DevTools → Emulate CSS media feature → `prefers-contrast: more`
- [ ] **Проверить:**
  - [ ] Пузыри с плотной заливкой (не прозрачные)
  - [ ] Толстые контуры/рамки
  - [ ] Читаемый текст на любом фоне

- [ ] Визуальный toast работает? ⬜ Да / ⬜ Нет
- [ ] Визуальный будильник работает? ⬜ Да / ⬜ Нет
